Para executar o bash, certifique-se de estar com um terminal aberto na pasta:
/decifra

Já na pasta, o seguinte comando devem ser utilizados para decifrar o código fornecido:
• Para decifrar:
– ./decifra < texto-cifrado.txt > texto-aberto.txt

Sendo cada parâmetro:
./decifra - Comando para executar o bash ligado a pasta
< texto-cifrado.txt - Arquivo de origem da frase criptografada
> texto-aberto.txt - Arquivo de destino da frase decodificada
